Myecommerce

## Technologies
* Spring Boot

## Setup
* All related maven dependencies should be imported
* mvn clean install -> at root
* mvn verify -> to jacoco check
* mvn spring-boot:run
* mvn test -> to run tests
* jacoco report is accesible after mvn test -> below target/jacoco-coverage-report
